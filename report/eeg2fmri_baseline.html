基于 HRF 感知残差条件流匹配的 EEG-to-fMRI ROI 预测基线

数学表述、逐步解释与批判性分析

本文将当前仓库中的实现整理为一个可放入论文的方法基线：HRF-aware Deterministic Mean + Residual Conditional Flow Matching。其核心思路是：先用 EEG 条件编码预测一个确定性均值，再对均值无法解释的残差使用条件流匹配建模，从而同时得到点预测和不确定性样本。

1. 问题定义与符号

X ∈ R^(n_ch × T), Y ∈ R^(L × R)

其中，X 是 EEG 输入，Y 是目标 ROI BOLD chunk；当前仓库默认使用 26 个 EEG 通道、8 个 TR 的 chunk 和 64 个 ROI。

T = T_ctx · f_s + (L − 1) · TR · f_s

这表示模型并不是用一个瞬时 EEG 采样点去预测一个 BOLD 采样点，而是使用一段较长的 EEG 窗口去预测一小段 BOLD 序列。这样做的原因是神经活动到 BOLD 之间存在血流动力学延迟。

通俗理解
模型在回答的问题不是“此刻 EEG 对应哪个 fMRI 值”，而是“在过去约 30 秒 EEG 背景下，接下来这一小段 ROI 级 BOLD 轨迹最可能长什么样”。

2. 整体 pipeline

图 1. EEG 经双分支编码和 HRF-aware 条件对齐后，先输出均值，再对残差做 conditional flow matching。
X → Z → C → μ → R* → Ŷ

这里，Z 是 EEG token，C 是带 HRF 对齐先验的条件表示，μ 是确定性均值预测，R* 是真实残差，Ŷ 是最终预测。

3. 从 EEG 到条件表示：X → Z → C

3.1 双分支编码

H^temp = Enc_temp(X) ∈ R^(N × d)
H^spec = Enc_spec(X) ∈ R^(N × d)
Z = Fuse( PE(H^temp + H^spec) ) ∈ R^(N × d)

含义：Enc_temp 负责时域波形特征，Enc_spec 负责 patch 级频谱能量特征。两者相加后再加位置编码，得到后续所有模块共享的 EEG token。

作用：把很长的 EEG 波形压缩成一串“局部时域+频域摘要”的 token，既保留节律结构，也保留局部动态。

批判性见解
这是合理的表征学习设计，但它仍然是经验性工程选择，而不是从 EEG-fMRI 生理耦合理论唯一推导出来的结构。

3.2 HRF-aware 条件对齐

p_n = (n · s + P / 2) / f_s
τ_l = T_ctx + (l − 1) · TR
Δ_(l,n) = τ_l − p_n
B_(l,n) = - (Δ_(l,n) − τ_peak)^2 / (2 σ_hrf^2), if 0 ≤ Δ_(l,n) ≤ τ_max B_(l,n) = -∞, otherwise
C = SA( CA(Q, Z; B) ) ∈ R^(L × d)

含义：每个待预测 BOLD 时间点都会去“看”一组 EEG token，但这种注意力不是完全自由的，而是被 HRF 偏置矩阵 B 约束。时差接近 HRF 峰值时，注意力得分更高；超过最大合理延迟时直接屏蔽。

作用：把“神经活动经过数秒延迟才体现在 BOLD 上”的先验编码进网络中，让模型更容易学到正确的时间对齐关系。

批判性见解
这一步是 HRF-aware，但不是严格的生理 HRF 卷积模型。它更像“带血流延迟先验的注意力”，而非个体化血流动力学估计。

4. 确定性主干：C → μ

μ = f_mean(C) ∈ R^(L × R)
L_mean = λ_mean · Huber(μ, Y) + λ_td · || Δ_t μ − Δ_t Y ||_1
Δ_t A = A_(2:L) − A_(1:L−1)

含义：μ 是在当前 EEG 条件下最可能的 ROI BOLD 序列中心估计。Huber 损失约束绝对数值误差，时间差分项约束上升、下降和变化趋势。

作用：先把最稳定、最容易预测的主趋势学出来，后续生成分支只需要处理“均值没解释掉的部分”。

5. 残差条件流匹配：μ → R* → R^

5.1 残差定义

R* = Y − sg(μ)

含义：R* 是真实目标相对于均值预测的剩余误差。sg 表示 stop-gradient，即数值上使用均值预测，但不让残差分支的梯度直接反向改写这一步误差分解。

作用：把整个任务拆成“先拟合主趋势，再拟合剩余不确定性”。

5.2 条件流匹配目标

ε ~ N(0, I), t ~ Unif(0, 1)
X_t = (1 − t) σ ε + t R*
U_t = dX_t / dt = R* − σ ε
L_cfm = λ_cfm · E_(t, ε) [ || v_θ(X_t, t, C) − U_t ||_2^2 ]

含义：模型并不直接回归真实残差，而是学习一条从噪声走到真实残差的路径。X_t 是路径上的中间状态，U_t 是理想速度，v_θ 则是模型学习到的条件速度场。

作用：让模型学习“在当前条件和当前路径位置下，残差应该往哪个方向变化”，而不是只学一个单点输出。

批判性见解
这里确实是 conditional flow matching，但它建模的是 残差分布，不是整个 BOLD 信号本体分布。因此更准确的叫法是“残差条件流匹配基线”，而不是“纯 EEG-to-fMRI CFM 模型”。

6. 推理：从条件表示得到最终预测

C = f_cond(X), μ = f_mean(C)
S_0^(m) = σ ξ^(m), ξ^(m) ~ N(0, I)
dS_τ^(m) / dτ = v_θ(S_τ^(m), τ, C), τ ∈ [0, 1]
R_hat^(m) = S_1^(m)
Y_hat^(m) = μ + R_hat^(m)
Y_hat = (1 / M) Σ_m Y_hat^(m)

含义：先得到条件均值 μ，再从高斯噪声出发沿 learned vector field 积分到终点，得到残差样本，最后加回均值得到最终样本预测。

作用：多次采样可以得到一组可能的 BOLD 轨迹，从而估计均值、方差、预测区间、CRPS 等概率指标。

7. 课程式训练

L = L_mean + L_cfm
L^(e) = L_mean, if e < e_0 L^(e) = L_mean + L_cfm, if e ≥ e_0

含义：前若干个 epoch 只训练均值头，待其稳定后再加入 flow 分支。

作用：避免一开始同时学习主趋势与残差分布导致优化不稳定。

8. 从 chunk 到整条 scan

Y_hat_scan(t) = (1 / K_t) · Σ_(k: t ∈ W_k) Y_hat^(k)(t)

含义：整条 scan 的某个时间点往往会被多个重叠 chunk 覆盖，因此对所有覆盖该时间点的 chunk 预测取平均。

作用：减少 chunk 边界效应，并恢复完整的 ROI 时间序列。

9. 这一基线在理论上是否可行

如果任务被限定为“从同步 EEG 预测低频、ROI 级、resting-state BOLD 动态”，那么这是一个可做的统计预测问题。因为 EEG 和 fMRI 测的是同一神经活动的不同代理变量，而 ROI 压缩、低频化和同步采集都显著降低了难度。

但如果把问题说成“仅凭 scalp EEG 唯一恢复全脑真实神经活动”，那是不成立的。原因在于 EEG 逆问题本身不适定，BOLD 也只是慢速、间接、带血流延迟的代理变量。

核心批判
这个仓库的方法在工程上是自洽而且值得作为 baseline 的；但它不是严格生理模型，也不是条件似然意义下完整的概率模型。论文中最好将它定位为“工程可复现的 HRF-aware residual CFM baseline”，而不是“神经血流耦合的最终答案”。

10. 它与 NeuroBOLT 的差异

  1. 本仓库是一个 HRF-aware residual CFM baseline，而不是 NeuroBOLT 原文方法本身。
  2. 本仓库从头训练 temporal + spectral encoder；NeuroBOLT 官方代码依赖预训练 LaBraM 表征。
  3. 本仓库显式分成“均值头 + 残差 flow 头”；NeuroBOLT 公开材料并未以 residual CFM 为核心。
  4. 本仓库将 HRF 先验写成固定高斯形状的 attention bias；NeuroBOLT 更强调多维特征映射而非这一具体结构。
  5. 本仓库更适合作为可运行、可消融的 compact baseline；NeuroBOLT 的定位更接近完整方法论文。

11. 建议在论文中如何使用这个 baseline

建议将该方法命名为 HRF-aware Deterministic Mean + Residual Conditional Flow Matching baseline，并配套以下消融：

  • 只保留均值头，检验概率建模是否真的有增益。
  • 将残差 CFM 替换为高斯 NLL 头，检验 flow 的必要性。
  • 去掉 HRF 偏置，检验显式时延先验是否有效。
  • 尝试 OT-CFM 或其他更强的路径设计，检验 vanilla CFM 是否已足够。
